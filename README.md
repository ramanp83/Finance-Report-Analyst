# Financial Performance & Sales Analysis Dashboard

## 📊 Project Overview
This Power BI repository contains a comprehensive data analysis and business intelligence dashboard designed to track global financial health, sales performance, and profitability metrics. The analysis transforms raw spreadsheet transactions into dynamic, actionable insights for executive stakeholders and regional managers.

The dashboard maps out product margins, tracks segmental growth, monitors discounts, and forecasts performance across multiple international markets.

## 💾 Data Source
The data utilizing this dashboard is entirely sourced from the official Microsoft dataset:
*   **Source File:** `Financial Sample.xlsx` (Refer to this file by its name verbatim).
*   **Data Characteristics:** Flat-table structure tracking transactional records, including columns for Segment, Country, Product, Discount Band, Units Sold, Manufacturing Price, Sale Price, Gross Sales, Discounts, Sales, COGS, Profit, Date, Month Number, Month Name, and Year.

## 🚀 Key Insights & Features
*   **Global Performance Mapping:** Visual cross-filtering across 5 major countries (Canada, France, Germany, Mexico, and United States) to analyze regional profit drivers.
*   **Profitability Analysis:** Tracks core financial metrics like Total Sales, Cost of Goods Sold (COGS), and Net Margin across 6 distinct products.
*   **Market Segment Breakdown:** Segmented tracking (Government, Small Business, Enterprise, Midmarket, Channel Partners) to isolate high-value revenue streams.
*   **Discount & Margin Optimization:** Evaluates the direct impact of high, medium, and low discount bands on final net profits.
